* 벨로퍼트(https://react.vlpt.us/) 리액트 강의 추천
* 실무에서는 함수형 컴포넌트를 더 많이 사용 (대세)
* Next.js도 할 줄 알면 좋다 (+타입스크립트도)
* 리액트에서는 구조 분해 할당을 할 줄 알아야함

[ 컴포넌트(Component) ]
- 리액트 앱을 구성하는 기본 단위
- 모든 컴포넌트의 이름은 반드시 대문자로 시작
- 함수 선언식, 표현식, 화살표 함수 전부 컴포넌트 정의 가능
**리액트는 리턴 시 루트(부모) 태그가 무조건 있어야 한다 (하나의 요소만을 리턴 할 수 있기 때문에)


[ 컴포넌트 속성(Props) ]
- 컴포넌트에 전달할 인자값(함수에 전달하는 인자와 비슷한 역할)
- 거의 전부 'props' 라는 인자 이름을 사용
- props 값은 수정이 불가능
- 함수를 값으로 취급하여 속성 값으로 함수나 메소드도 전달 가능
- 구조 할당 분해 방법을 많이 사용한다
