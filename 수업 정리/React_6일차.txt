[ useState 훅 ]
- 저장할 수 있는 자료형에 제약이 없으므로 객체 저장도 가능
- 객체의 값을 변경 시 '기존 객체 내용을 복사하고 새롭게 갱신하는 속성값을 덮어쓰는 방식(overwrite)을 사용'
- 중첩 객체의 경우, 중첩 객체의 내용을 복사하며 새롭게 값을 갱신하도록 작성
- 세터 함수 호출하는 과정에서 이전 값을 참조하여 값 수정이 필요한 경우, 이전 상태 값을 전달받는 콜백 함수를 전달 가능


[ 배열 리스트 ]
- concat(추가), map(변경), filter(삭제) < 원본 리스트에 영향을 끼치지 않도록(불변성)해야하니 이러한 메서드들을 사용
- React는 내부적으로 상태값을 비교할 때 같은 객체인지 여부('참조'가 같은지)를 이용하여 상태 변경 및 컴포던트 함수 호출
